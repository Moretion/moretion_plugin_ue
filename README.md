# moretion_plugin_ue
魔迅动捕配套ue插件

## 1. ue软件准备
此插件基于ue5.3引擎编写
ue官方下载链接：https://www.unrealengine.com/zh-CN/download

## 2. 插件下载
克隆此项目根据自己需求选择使用MotionSuit_ue5.3还是MotionSuit_ue5.4

## 3. 插件安装说明
1. 先下载 Epic Games 虚幻引擎，在 Epic 中找到 UE 引擎下载，安装引擎时需注意不要勾选
引擎源代码选项。
